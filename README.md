<a name="how-it-works"></a>üîß How It Works
<h3>Processing Pipeline</h3> <pre> 1. USER INPUT ‚îî‚îÄ> Text / URL / Video / Image 2. PREPROCESSING ‚îú‚îÄ> Text: Direct input ‚îú‚îÄ> URL: Content extraction ‚îú‚îÄ> Video: Audio extraction ‚Üí Transcription (simulated) ‚îî‚îÄ> Image: OCR extraction (simulated) 3. PARALLEL API PROCESSING (Simultaneous) ‚îú‚îÄ> Sapling AI Detection API ‚îÇ ‚îî‚îÄ> Returns: Overall score, Sentence scores ‚îÇ ‚îú‚îÄ> Grok AI (xAI) ‚îÇ ‚îî‚îÄ> Returns: Claim verification, Sources, Status ‚îÇ ‚îî‚îÄ> Perplexity API ‚îî‚îÄ> Returns: Research summary, Real-time sources 4. AGGREGATION ‚îî‚îÄ> Combine results into unified data structure 5. DISPLAY ‚îî‚îÄ> Render comprehensive results page </pre> <h3>Why Parallel Processing?</h3> <p><strong>Speed:</strong> Instead of waiting 15 seconds sequentially (5s + 5s + 5s), all APIs run at once = ~5-7 seconds total.</p> <pre><code> // Parallel execution const [aiResult, factResult, realInfo] = await Promise.all([ analyzeWithSapling(text), verifyFactsWithGrok(text), getRealInformationFromPerplexity(text) ]); </code></pre> <h3>AI Detection Algorithm</h3> <ol> <li><strong>Text Analysis:</strong> Sapling API analyzes linguistic patterns</li> <li><strong>Score Calculation:</strong> Returns 0-1 probability (converted to 0-100%)</li> <li><strong>Sentence Breakdown:</strong> Each sentence gets individual score</li> <li><strong>Model Attribution:</strong> Determines likely AI model: <ul> <li>Score &gt; 0.7 ‚Üí GPT-4</li> <li>Score &gt; 0.4 ‚Üí GPT-3.5</li> <li>Score ‚â§ 0.4 ‚Üí Human</li> </ul> </li> <li><strong>Metrics Calculation:</strong> <ul> <li><strong>Perplexity:</strong> <code>15 + (score * 50)</code></li> <li><strong>Burstiness:</strong> Variance of sentence scores</li> </ul> </li> </ol> <h3>Fact Verification Algorithm</h3> <ol> <li><strong>Claim Extraction:</strong> Split text into sentences, filter for factual statements (min 20 chars)</li> <li><strong>Claim Verification:</strong> Each claim sent to Grok AI with prompt: <pre><code>"Verify this claim and provide sources in JSON format"</code></pre> </li> <li><strong>Status Assignment:</strong> Grok returns one of: <ul> <li><strong>verified:</strong> Confirmed by 3+ reliable sources</li> <li><strong>partially-true:</strong> Mixed evidence</li> <li><strong>false:</strong> Contradicted by credible sources</li> <li><strong>unverifiable:</strong> Insufficient data</li> </ul> </li> <li><strong>Credibility Scoring:</strong> <ul> <li>Verified = 100 points</li> <li>Partially true = 50 points</li> <li>False = 0 points</li> <li>Unverifiable = 50 points</li> <li><strong>Overall = Average of all claims</strong></li> </ul> </li> <li><strong>Bias Detection:</strong> Keyword matching for political/sensational language</li> </ol> <h3>Real Information Gathering</h3> <ol> <li><strong>Query Building:</strong> Create research query based on content and category</li> <li><strong>Perplexity Search:</strong> Real-time web search using <code>sonar-pro</code> model</li> <li><strong>Response Parsing:</strong> Extract: <ul> <li>Summary (first 3 sentences)</li> <li>Key facts (bullet points)</li> <li>Source URLs (up to 5)</li> <li>Expert insights (final sentences)</li> </ul> </li> <li><strong>Source Credibility:</strong> Assign scores (85-100%) based on domain authority</li> </ol>
